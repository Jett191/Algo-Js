# 排序

### 选择排序

- 可以简单地把一个数组分为两个部分，一个未排序区间，一个为已排序区间
- 通过循环不断的去缩小未排序的区间
- 每轮排序都将未排序区间的最小值放到已排序区间的最后
- 从左至右依次扩大已排序的区间，相应的未排序区间会减小

### 冒泡排序

- 想象最大的元素就是最大的泡泡，数组最右边就是水面
- 从左至右将相邻的两个泡泡进行比较，大的泡泡就往水面方向移动冒出
- 依次从右至左比较相邻的两个泡泡，最后最大的泡泡将移至水面
- 随后再从头开始找出第二大的泡泡，重复过程
